üîê Hash Functions

Provides implementations of secure one-way functions like SHA-2, SHA-3, and BLAKE2b/3.

local hash = require("SHA2_256")
local digest = hash:Digest("hello world")
print(digest)

üîí Encryption

Symmetric AES-based encryption with secure padding and IV generation.

local aes = require("AES")
local ciphertext = aes:Encrypt("secret", key)
local plaintext = aes:Decrypt(ciphertext, key)

üîë Key Management

Use securely derived keys. Consider using PBKDF2 or HKDF for password-based secrets.

Never store plain keys. Always zero buffers after use.

üìò API Reference

Each module uses PascalCase naming and returns a class-like table. Example:

local sha3 = require("SHA3_256")

-- API
sha3:Digest(data: string): string
